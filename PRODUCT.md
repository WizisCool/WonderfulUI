# Product

## Register

product

## Users

Valorant 玩家,通过腾讯 ACLOS / 掌上无畏契约客户端录制"无畏时刻"(highlight),日常在游戏对局后回看自己的高光。目标用户的电脑常驻 ACLOS,本地已经堆积了数十~数百个 highlight 视频,但完整回放体验强依赖客户端启动 + 登录 + 联网。

## Product Purpose

让用户在**不启动 Valorant / ACLOS 客户端**的情况下离线浏览自己的高光时刻目录,做到:
- 按地图 / 英雄 / 胜负 / 击杀 / 日期快速定位想要的某场对局。
- 直接调用系统播放器播放本地 mp4(不动游戏、不联网、不写 ACLOS 任何文件)。
- 维护"严格只读 + 路径可移植"两条铁律,不破坏游戏客户端、不触发 ACE / Vanguard。

成功定义:用户在自己常用的任意一台 Windows 电脑上解压即用,无需配置;首次启动能看到所有账户的高光列表;点击播放能在 3 秒内出画面。

## Brand Personality

**桌面 · 纯净 · 效率**(用户原文)。参考系:Steam 库的"列名 + 缩略图 + 单击即看"、Riot Client 的暗色调战术感、VLC 那种"工具感"。

气质具体化:
- 桌面优先:键盘 + 鼠标双修,焦点环清晰,常用操作可纯键盘完成。
- 纯净:不堆装饰,一个画面只说一件事。封面是真实数据,不是渐变占位。
- 效率:列表 → 详情 → 播放 三步完成,中间不弹"加载中"等无意义页面。

## Anti-references

- **AI 量产 SaaS 风**:紫蓝渐变 + 卡片 + hero 大字 + 玻璃拟态 → 直接拒绝,这是 2024 default 模板。
- **Windows 资源管理器风**:灰 panel + 8pt 网格 + 锐利边框 → 太工具化,没有"看自己的战绩"的仪式感。
- **Steam / Epic 启动器风**:封面 + 购买 + 评分 + 愿望单 → 这是商业产品,本工具是私人档案。
- **交互动画过载**:page transition / hover ripple / 进入动画 → 用户来看自己的东西,不需要"演示"。
- **Glassmorphism 默认**:仅在确实增强层级(比如浮在视频之上的控制条)时用,不作主调。

## Design Principles

1. **Show, don't decorate.** 列表项 = 真实封面 + 真实数据(击杀 / 地图 / 时间)。不用渐变图标、空状态插画、抽象几何代真东西。
2. **Three clicks, in-app.** 列表 → 详情 → 内置播放(居中模态,YouTube 风格控制条)。系统播放器仅保留为右键降级入口。模态只用于播放器(聚焦观看)和不可逆操作(如:删除本地视频)。
3. **Read the room.** 暗色优先(电竞 / 长时间观看),但**不能**做成纯黑;底色用一个很深的暖灰(带品牌色相的痕迹),让红色高光自然跳出来。
4. **Native over webby.** 视频用内置 WebView2 播放器播放(H.264 1440×1080 60fps,硬件加速,极简模态壳)。系统播放器作为右键 fallback 始终可用。
5. **Forgiving, not paranoid.** NAS 未挂载、ACLOS 在写文件、mp4 被搬走——优雅降级 + 一句人话提示,不弹 stacktrace。

## Accessibility & Inclusion

- WCAG 2.1 AA:文本与背景对比度 ≥ 4.5:1;大文本(≥18px 或粗体 ≥14px)≥ 3:1。
- 焦点环:键盘 Tab 聚焦时必须有可见的 2px 焦点指示器,不能只有颜色变化。
- 不仅靠颜色:胜负用图标 + 颜色双重编码(色盲友好)。
- 键盘可达:列表项可回车播放,详情面板可 Esc 关闭,搜索框可 Esc 清空。
- 动效由应用自管:不跟随 Windows「动画效果」/`prefers-reduced-motion`;保持短过渡(约 80–260 ms),不在各组件散落 reduce 覆盖。
- 字号:基础 14px(在 1280×800 窗口下),用户可在设置里调到 18px。
